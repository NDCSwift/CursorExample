# 🖱️ Cursor Example — Building SwiftUI with Cursor AI

A SwiftUI quotes app built using Cursor — demonstrating how to use AI-assisted coding in the Cursor IDE to rapidly build iOS features.

---

## 🤔 What this is

CursorExample shows the workflow of using Cursor (an AI-powered code editor) to build a SwiftUI app from scratch. The app is a quotes viewer with a login flow — but the real focus is watching Cursor generate and iterate on SwiftUI code with AI assistance, showing where it shines and where you still need to guide it.

## ✅ Why you'd use it

- **See Cursor in action** — a real example of AI-assisted SwiftUI development in the Cursor IDE
- **Complete app structure** — `LoginViewModel` for auth state, `QuoteView` for browsing content
- **Practical AI coding workflow** — shows how to prompt, review, and refine AI-generated Swift code
- **Before/after reference** — useful starting point for using Cursor on your own SwiftUI projects

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/CursorExample.git
cd CursorExample
```

### 2. Open in Xcode
Double-click `CursorExample.xcodeproj`.

### 3. Set Your Development Team
TARGET → Signing & Capabilities → Team

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique identifier.
